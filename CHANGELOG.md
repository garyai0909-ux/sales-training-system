# CHANGELOG

## v2.0.0 - 2026-02-09

### 新增
- 全新主管後台 KPI 儀表：總訓練次數、參訓人數、平均分數、達標率
- 成績分析：常錯題 Top5、能力維度弱項、近 14 日學員趨勢
- 題庫管理 CRUD：新增 / 編輯 / 刪除題目
- 題庫驗證：檢查 id 重複、題目空值、選項數量、正解是否存在
- 匯出功能：CSV + XLSX（多工作表）

### 改善
- 手機版體驗與整體 UX：更易觸控、排版更清楚、按鈕與卡片一致化
- 錯題複訓模式：整合歷史錯題池，答錯題可再加入本輪複訓
- 題目加入能力維度（dimension）供分析使用
- 成績記錄增加 dimensionStats 與 wrongIds，便於主管追蹤

### 穩定性/錯誤處理
- localStorage 讀寫加入保護
- 題庫載入時做正規化與容錯
- 清空資料、刪題、重置題庫皆加入確認流程
